== Description ==

DP Tweaks gives you more controls for built-in features of WordPress, and combines several useful WordPress tweaks pertaining to performance, security, admin and more.

DP Tweaks is a powerful yet easy-to-use plugin, it specializes in simple toggle-on/off changes, and works by adding a "DP Tweaks" page to your WordPress options, all the options are accessible from one single settings page.

= Features =

**Admin**

Gives you more power to control the wordpress administration panel and improving usability.

* Disable Autosave
* Disable Post Revision
* Disable the Flash uploader
* Disable WordPress from sending pings to your own site
* Show IDs in admin edit view
* Enable both Trash and Delete Permanently at the same time

**Security**

Useful security tweaks to protect your WordPress-powered blog.

* Remove WordPress version in anywhere
* Remove Update informatioin for non-admin
* Hide Login error message
* Protect WordPress against malicious URL requests


**Performance**

Speed up your WordPress blog, increase WordPress performance.

* Use common javascript libraries from Google's AJAX Libraries CDN, by [Use Google Libraries](http://wordpress.org/extend/plugins/use-google-libraries/).

**More**

In the future releases, DP Tweaks will add more new tweaks. These new tweaks from you, any suggestions and ideas would be grateful!

Enjoy it now and begin tweaking!

== Installation ==

1. Upload the entire folder `dp-tweaks` to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Change Settings on 'Settings > DP Tweaks'.
